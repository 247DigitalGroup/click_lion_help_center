                                          Report Interface

                                          Access report interface

                                          Report is an important section in ClickLion. It is the performance insight for advertisers and publishers to improve their performance in the future. To access Report section, please follow these steps:

                                          1. Sign in to your ClickLion account.

                                          2. Click Report section on top navigation.

                                          Report Interface

                                          At first, you will see a blank report with a box to choose the report source entity (Learn more about source entity) and a navigation menu on the left. In the left menu, we have General and Geo report options.

                                          General Report

                                          General report will provide you stats (in many metrics) of a source entity in a specified date range with a specified frequency. To get a report:

                                          1. Make sure you are in General Report section

                                          2. Choose a source entity in the box which displays Select report source

                                          3. If there is no data for that source entity, you will see a notice saying No report data for this time frame, .... Otherwise, you should see the report for selected source entity.

                                            In the report, there are three main parts: main graph, metric graphs and data table. Main graph is a detailed graph for selected metric. Metric graphs is a set of graph that displays many graphs for each metric, you can click one graph to bring them to the Main graph for better view. Data table contains all data for the report which is used to build up those graphs. In that table, you will see many columns that represent metric stats for each time period (Learn more on metrics).

                                          You can change date range and frequency of the report. Date range picker can be found under the source entity box on the left, frequency selector is on the right.

                                          Geographic report

                                          General report will provide you stats (in many metrics) on a source entity in a specified date range for for many countries. To get a report:

                                          1. Make sure you are in General Report section

                                          2. Choose a source entity in the box which displays Select report source

                                          3. If there is no data for that source entity, you will see a notice of No report data for this time frame, .... Otherwise, you should see the report for selected source entity.

                                            The geographic report is basically the same with General report except there is no frequency, everything is calculated on country-wised basis. You can see country entries in the data table instead of a period.

                                          Understand your reports

                                          Metrics and how ther are related

                                          Report refers to report section on the top navigation bar. It also refers to the performance report for a source entity (a campaign, an article, a domain or a creative) in a specific date range with different frequencies (hourly, daily or monthly). A report contains many statistics on many metrics and intuitive graphs for advertisers and publishers to track their performance conveniently.

                                          Report source (your campaign, articles, domain, creative)

                                          Report source is the entity you want to get report for. It can be a campaign, an individual article, your registered domain or a creative.

                                          campaigns, domains and creatives are tracked by ClickLion report system. However, articles are not tracked by default.
